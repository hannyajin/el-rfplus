REST JSON Overview:

GET /api/v1/stores

Description: Lists all stores.
Format: [Store] // See Store Format below


GET /api/v1/stores/:id
Example: api/v1/stores/54574297e4b02a012c93a6eb

Description: A single store.
Format: Store
  {
    _id: String - Identifies this store. Used as a reference in ALL orders.

    name: String - name of store

    address: { // example: 871 Younge st. Toronto ON CA M4W4H2
      store: String - name of store
      number: String // 871
      street: String //Younge st.
      town: String // Toronto
      state: String // ON CA
      zip: String // M4W4H2
    }

    phone: String, // +9872213364
    fax: String, // +9872213648
    email: String,  // gnt.pharma@gmail.com

    logo: {
      href: String // url to logo
    }

    gps: { // gps coordinate for store
      lat: String, // 43.674043,
      lon: String // -79.388187
    },

    updatedAt: last updated date
    createdAt: creation date
  }

## callDoctor



POST /api/v1/orders/callDoctor

Request:
  body:
  {
    // User Information
    user: {
      name: {
        first: String,
        last: String
      },
      phone: String,
      email: String,

      allergies: [String], // optional
      medicalConditions: [String] // optional
    },

    // Doctor Information
    doctor: {
      name: {
        first: String,
        last: String
      },
      phone: String, // optional
      address: String, // optional
      city: String
    },

    // Prescriptions
    prescriptionNames: [String],

    // Pharmacy
    store: String, // see _id field in /api/v1/stores

    instructions: String // optional
  }



## sendPrescription


POST /api/v1/orders/sendPrescription

Request:
  body:
  {
    // Photo of prescription
    prescription: {
      name: String, // optional
      type: String, // optional
      data: Buffer, // img data as binary
      stringData: String // img data as string, optional
    },

    // User Information
    user: {
      name: {
        first: String,
        last: String
      },
      phone: String,
      email: String,

      allergies: [String], // optional
      medicalConditions: [String] // optional
    },

    // Prescriptions
    prescriptionNames: [String],

    // Pharmacy
    store: String, // see _id field in /api/v1/stores

    instructions: String // optional
  }
