REST Methods:

GET /api/v1/stores
GET /api/v1/stores/:id

Example: http://localhost:3000/api/v1/stores/54574297e4b02a012c93a6eb



## callDoctor



POST /api/v1/orders/callDoctor

Request:
  body:
  {
    // User Information
    user: {
      name: {
        first: String,
        last: String
      },
      phone: String,
      email: String,

      allergies: [String], // optional
      medicalConditions: [String] // optional
    },

    // Doctor Information
    doctor: {
      name: {
        first: String,
        last: String
      },
      phone: String, // optional
      address: String, // optional
      city: String
    },

    // Prescriptions
    prescriptionNames: [String],

    // Pharmacy
    store: String, // see _id field in /api/v1/stores

    instructions: String // optional
  }



## sendPrescription


POST /api/v1/orders/sendPrescription

Request:
  body:
  {
    // Photo of prescription
    prescription: {
      name: String, // optional
      type: String, // optional
      data: Buffer, // img data as binary
      stringData: String // img data as string, optional
    },

    // User Information
    user: {
      name: {
        first: String,
        last: String
      },
      phone: String,
      email: String,

      allergies: [String], // optional
      medicalConditions: [String] // optional
    },

    // Prescriptions
    prescriptionNames: [String],

    // Pharmacy
    store: String, // see _id field in /api/v1/stores

    instructions: String // optional
  }
